# Import your txns

**There’s several ways you can** import your txns into **Acckenn Calc:**

1. Import transaction history (.csv) **from exchanges**
2. **Manual import** transaction by Acckenn Calc form
3. Load transaction history (.csv) **saved from Acckenn Calc**

### Supported Exchanges

Currently, we supports the following **exchanges:**

- **Bitkub** -> [How to export txn history](https://support.bitkub.com/hc/en-us/articles/360000338012-FAQ-Where-can-I-export-my-transaction-history-)
- **Satang Pro** -> [How to export txn history](https://satang.zendesk.com/hc/en-us/articles/6285729839257-How-to-export-coin-transactions-history)
